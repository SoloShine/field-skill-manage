向导式分步弹窗

分 3 步引导：选源 Agent → 勾选 Skills → 确认并处理冲突。每步聚焦一个决策，清晰不易出错。

Pros

  • 逐步引导，不易遗漏
  • 冲突处理可在第 3 步集中展示

Cons

  • 步骤多，操作路径长
  • 弹窗内容需切换

Cons

  • Skill 多时弹窗可能较长

拖拽面板

左右双栏布局，左栏展示源 Agent Skills，右栏展示目标。支持拖拽或点击添加。

Pros

  • 交互直观，所见即所得
  • 双栏对比清晰

Cons

  • 拖拽实现复杂度高
  • 冲突提示不好融入
  • 批量操作不如勾选方便
